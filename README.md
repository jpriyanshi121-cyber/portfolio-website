# 🚀 Priyanshi's Portfolio Website

A modern, interactive portfolio website built with **React**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**. Showcasing projects, skills, education, and community involvement with smooth animations and a sleek dark theme.

🌐 **Live Demo:** [https://portfolio-website-azure-iota-91.vercel.app](https://portfolio-website-azure-iota-91.vercel.app)

---

## ✨ Features

- **Responsive Design** - Optimized for mobile, tablet, and desktop
- **Smooth Animations** - Beautiful transitions powered by Framer Motion
- **Dark Theme** - Eye-friendly gradient background with glass-morphism effects
- **Interactive Navigation** - Smooth scroll navigation with mobile menu
- **Sections Included:**
  - Hero section with call-to-action buttons
  - About me with key statistics
  - Education timeline
  - Hackathon projects & achievements
  - Community involvement
  - Technical skills & tools
  - Contact section
- **Project Links** - Direct links to Figma prototypes and GitHub repositories

## 📦 Installation

### Prerequisites
- Node.js 16+ 
- pnpm (recommended) or npm

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/jpriyanshi121-cyber/portfolio-website.git
   cd portfolio-website
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   ```

3. **Start development server**
   ```bash
   pnpm dev
   ```

4. **Open in browser**
   Navigate to `http://localhost:5173`
